Sistema de gerenciamento de estoque: este projeto envolve a criação de um banco de dados para gerenciar o estoque de uma loja. O banco de dados pode armazenar informações sobre os produtos, incluindo seu nome, descrição, preço e quantidade em estoque. O sistema também pode permitir que os usuários adicionem novos produtos, atualizem informações existentes e gerem relatórios de vendas.
